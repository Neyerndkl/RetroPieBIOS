# RetroPieBIOS

BIOS files collection for [RetroPie](https://retropie.org.uk) (currentlly updated for **version 4.6**).

[RetroPie's Docs](https://retropie.org.uk/docs/BIOS/) says:

    Supported bios files for the libretro cores can be scanned using the official Bios.dat from Libretro-database.

The file `Bios.dat` was renamed to [System.dat](https://github.com/libretro/libretro-database/blob/master/dat/System.dat), in which lists `System, firmware, or BIOS files used by libretro`.

With google's help **ALL 189 FILES WERE FOUND AND COLLECTED** from the web and saved into the folder `bios` and all file md5 hash values has been checked.

Copy all these files into your RetroPie's bios folder and enjoy your games! ðŸ˜‰
